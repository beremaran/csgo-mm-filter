# Counter Strike Global Offensive Matchmaking Relay Filter

Simply blocks any other matchmaking relays other than your choice of location.

```
Usage: csgo_server_blocker [options] [command]

Options:
  -V, --version  output the version number
  -v, --verbose  verbose mode
  -h, --help     output usage information

Commands:
  only <region>  Only be able to connect <region> relays.
  revert         Revert firewall configuration
  list           List available regions
  interactive    Interactive mode
```
